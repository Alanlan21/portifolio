2) NestJS-Attack-and-Defense-Lab

3) Fala-Pai

- Destaques: propósito social real, síntese de voz de alta qualidade, PWA instalável, UX bem cuidada.
- Status: finalizado/estável (deploy ativo).

4) Calote-bot
- Tipo: bot de automação de cobrança via WhatsApp.
- Stack: Node.js, venom-bot, dayjs, inquirer, node-schedule.
- Destaques: templates dinâmicos com variáveis, agendamento, CLI interativo, documentação com personalidade.
- Status: finalizado/estável.

Arquitetura proposta do portfolio
- Landing de impacto com estética de terminal (sem clichê).
- Área principal em formato docs:
  - Sidebar de navegação
  - Conteúdo principal (MDX)
  - TOC na direita
- Páginas:
  - / (landing)
  - /sobre
  - /projetos
  - /projetos/[slug]
  - /contato
- Conteúdo dos projetos em MDX para facilitar evolução.

Stack técnica do portfolio
- Next.js (App Router) + React + TypeScript
- Tailwind CSS + shadcn/ui
- MDX para conteúdo
- Shiki para syntax highlight
- Motion para animações pontuais
- next-themes para dark/light
- (Opcional depois) Pagefind para busca Cmd/Ctrl+K
