## About the Program

Graduated in **Systems Analysis and Development** from the University of Fortaleza (Unifor), with emphasis on software engineering, systems architecture, and development best practices. Completed in December 2025 with a **GPA of 9.2**.

The curriculum combines solid theory (algorithms, data structures, databases, networking) with practical projects that simulate the complete software development lifecycle. The primary language used in the program was **Java**, the foundation for OOP, data structures, and applied project courses.

## Highlights

### UniMenu at UniforHub

**UniMenu** was the project that grew the most during undergrad: a complete ordering system for UNIFOR's cafeterias (NestJS + React + React Native). With faculty support, we brought the project to **Unifor Hub** — the university's innovation and startup hub, elected the Best Innovation Environment in Ceará — participating in the **O-HACKA-TA-ON hackathon in 2025**.
