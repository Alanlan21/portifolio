## Sobre o curso

Bacharelado em **Análise e Desenvolvimento de Sistemas** na Universidade de Fortaleza (Unifor), com ênfase em engenharia de software, arquitetura de sistemas e boas práticas de desenvolvimento.

A grade combina teoria sólida: algoritmos, estruturas de dados, banco de dados, redes. Com projetos práticos que simulam o ciclo completo de desenvolvimento de software.

## Destaques

### Projetos práticos

- **UniMenu:** sistema completo de pedidos para restaurantes universitários (NestJS + React + React Native), desenvolvido como projeto de conclusão de curso
- Projetos de banco de dados relacionais com modelagem ER e otimização de queries
- Implementação de APIs REST do zero com autenticação e autorização

### Disciplinas relevantes

- Engenharia de Software: ciclo de vida, padrões de projeto, SOLID
- Arquitetura de Sistemas: microsserviços, comunicação assíncrona
- Segurança da Informação: criptografia, OWASP, análise de vulnerabilidades
- Infraestrutura: redes, computação em nuvem, containers

### Formação complementar

Durante a graduação, busquei complementar a formação com experiências práticas reais, trabalhando como desenvolvedor profissional desde o segundo semestre, o que permitiu aplicar os conceitos acadêmicos diretamente em sistemas de produção.
