## Sobre o curso

Graduado em **Análise e Desenvolvimento de Sistemas** pela Universidade de Fortaleza (Unifor), com ênfase em engenharia de software, arquitetura de sistemas e boas práticas de desenvolvimento. Concluí o curso em dezembro de 2025 com **média escolar de 9.2**.

A grade combina teoria sólida (algoritmos, estruturas de dados, banco de dados, redes) com projetos práticos que simulam o ciclo completo de desenvolvimento de software. A linguagem principal adotada no curso foi **Java**, base para as disciplinas de POO, estruturas de dados e projetos aplicados.

## Destaques

### UniMenu no UniforHub

O **UniMenu** foi o projeto que mais tomou forma durante a graduação: um sistema completo de pedidos para as cantinas da Unifor (NestJS + React + React Native). Com o apoio de professores, levamos o projeto ao **Unifor Hub**, o hub de inovação e startups da universidade eleito o Melhor Ambiente de Inovação do Ceará, participando do hackathon **O-HACKA-TA-ON em 2025**.

### Outros projetos práticos

- Projetos de banco de dados relacionais com modelagem ER e otimização de queries
- Implementação de APIs REST do zero com autenticação e autorização
- Projeto multiplataforma com React Native entregue em produção

### Disciplinas relevantes

- Projeto de Arquitetura de Sistemas
- Técnicas de Integração de Sistemas
- Programação Orientada a Objetos
- Estrutura de Dados
- Desenvolvimento de Plataformas Móveis
- Segurança no Projeto de Software
- Teste e Validação de Sistemas

### Formação complementar

Durante a graduação, trabalhei como desenvolvedor profissional desde o segundo semestre: primeiro como estagiário na Dataged com Java/Spring, depois como desenvolvedor na Energia Pecém. Isso permitiu aplicar os conceitos acadêmicos diretamente em sistemas de produção.
